# Prework Study Guide Webpage

## Description

This Prework Study Guide was created to for students doing prework study

N/A

## Usage

To use the Prework Study Guide right click on the browser and select the inspect option or pressing Command+Option+I (macOS) or Control+Shift+I (Windows). A list of topics to study should appear in the console together with a suggestion of a topic to study.
